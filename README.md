# Algorithmic Order-Book Liquidity & Market Impact Engine

## Overview
Independent quantitative finance research project investigating market microstructure, order-book liquidity, and market impact using statistical analysis, real-time data engineering, and algorithmic modeling.
The project will begin with exploratory data analysis and literature review in R, followed by implementation of real-time order-book processing, liquidity metrics, and market impact simulation in Python.

## Objectives
- Research market microstructure and liquidity modeling
- Analyze historical and live Level II order-book data
- Compute liquidity and execution quality metrics
- Simulate market impact of large trades
- Develop an interactive dashboard for transaction cost analysis

# Project Phases
